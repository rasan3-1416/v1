About Me

Hi, Myself Md. Rakib Hasan, I am a skilled Frontend Developer who likes to build stuff that views the web. My interest in Frontend development started during my journey while learning UI/UX Design back in 2020. I felt very interested in making my designs more interactive with users which led me to Frontend development. After a while, I started learning the way of the web and started my career path as a Front-End Developer.

Fast forward to today, I had the privilege of working at many remote marketplaces like Fiverr, Freelancer.com, Upwork etc. During my journey through the web, my main focus was to build accessible, inclusive sites with complex functionality and security. Which led me to the backend part of the web.

So I have also started my journey for the Backend. To learn how the web works from the core and how complexity works from the view of a developer. Now I started learning React & Node.js.
